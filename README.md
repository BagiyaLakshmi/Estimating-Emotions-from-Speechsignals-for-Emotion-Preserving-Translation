# Estimating Emotion from Speech Signals for Emotion Preserving Translation. 

## 1. Conda env setup 

```
conda create -n pycp -y python=3.10.10             
```

```
conda activate pycp
``` 

## 2. Jupyter Notebook 

```
pip install jupyterlab
```

```
jupyter lab
```
